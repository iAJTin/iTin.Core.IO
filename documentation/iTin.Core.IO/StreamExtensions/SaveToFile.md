# StreamExtensions.SaveToFile method

Saves this stream into a file with name specified by parameter *fileName*. You can indicate whether to automatically create the destination path if it does not exist. By default it will try to create the destination path. The use of the ~ character is allowed to indicate relative paths, and you can also use UNC path.

```csharp
public static IResult SaveToFile(this Stream stream, string fileName, SaveOptions options = null)
```

| parameter | description |
| --- | --- |
| stream | Stream to save |
| fileName | Destination file path. Absolute or relative (~) paths are allowed |
| options | Output save options |

## Return Value

A IResult object that contains the operation result
